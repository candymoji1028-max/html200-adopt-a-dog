# [Adopt‑A‑Dog](https://candymoji1028-max.github.io/html200-adopt-a-dog/)

## **Description**
This project is part of the UW Front‑End Development Certificate program.
It is a multi‑page website built using semantic HTML and CSS to simulate an online dog adoption experience.

Site Synopsis:  
The Adopt‑A‑Dog site includes several pages that introduce users to adoptable dogs, provide blog content, and offer a simple checkout form. The focus of this project is on clean structure, accessibility, and responsive layout.

Pages Included:  
- Home Page – Introduction and featured dogs.  
- Dogs Page – A gallery of adoptable dogs with names, images, and descriptions.  
- Blog Page – Articles related to dog care and adoption tips.  
- Checkout Page – Simple form for completing an adoption request.  

## **Features** 
- Semantic HTML structure  
- Responsive layout tested on multiple devices  
- Cross-browser testing (Chrome, Edge, Safari on iPhone)  
- Consistent navigation across all pages  
- Mobile-friendly layout using media queries  
- Accessible labels and form fields  
- Alt text for images  
- Clean, readable typography and spacing
